# E-Commerce Customer Behaviour Analysis

## Project Overview
End-to-end data analysis project on a UK-based online retail dataset to identify high-value customers, detect churn risk, and uncover revenue trends using RFM segmentation.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite) — 5 analytical queries (see analysis_queries.sql)
- Power BI

## Dataset
UCI Online Retail II Dataset — 781,717 clean transactions, 
5,860 customers, 43 countries (Jan 2010 to Dec 2011)

## Key Findings
- 1,801 Champions generate 76% of total revenue (GBP 13.2M)
- 1,107 At Risk customers represent GBP 2.1M recovery opportunity
- UK dominates revenue at 82% of total sales
- November is consistently the peak revenue month

## Project Structure
- `retail.py` — Full Python code (cleaning, EDA, RFM, SQL)
- `data/` — Cleaned datasets and RFM output
- `sql/` — SQL query results
- `charts/` — EDA visualisations
- `ECommerce_Customer_Behaviour_Analysis_Report.docx` — Full project report
